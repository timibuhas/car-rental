# Car-Rental Management Application

Aceasta este o aplicație dezvoltată ca proiect universitar pentru gestionarea unei firme de închirieri auto. Aplicația permite administrarea flotei de mașini, rezervărilor și a clienților.

---

## Tehnologii utilizate
- **Backend:** ASP.NET Core, Entity Framework  
- **Frontend:** React  
- **Bază de date:** PostgreSQL  

---

## Funcționalități principale
- Gestionarea mașinilor și a caracteristicilor acestora  
- Crearea și urmărirea rezervărilor  
- Administrarea clienților și istoricul închirierilor  
- Dashboard interactiv pentru vizualizarea activității firmei  

---

## Ce am învățat
- Structurarea unei aplicații full-stack și separarea clară între frontend și backend  
- Crearea și gestionarea unei baze de date relaționale cu PostgreSQL  
- Folosirea Entity Framework pentru operațiuni CRUD și mapare obiecte 
- Dezvoltarea de interfețe responsive și interactive cu React  
- Implementarea fluxurilor de date și comunicarea între client și server
